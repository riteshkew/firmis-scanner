## Usage

```bash title="Terminal"
firmis policy validate <policy-file>
firmis policy check [path] --policy <policy-file>
```

## Subcommands

### `policy validate`

Validate a policy YAML file for syntax and schema correctness.

| Flag | Type | Description |
|---|---|---|
| `--verbose` | boolean | Show detailed validation output |

### `policy check`

Run a scan and evaluate results against a policy.

| Flag | Type | Default | Description |
|---|---|---|---|
| `--policy <file>` | string | — | Policy file to enforce |
| `--platform <name>` | string | auto-detect | Scan specific platform |
| `--format <type>` | enum | `json` | Output format |
| `--output <file>` | string | — | Save results to file |
| `--fail-on-violation` | boolean | `true` | Exit non-zero on policy violations |
