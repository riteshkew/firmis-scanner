> List detected AI agent platforms in your project.

## Usage

```bash title="Terminal"
firmis list [options]
```

## Options

| Flag | Type | Default | Description |
|---|---|---|---|
| `--json` | boolean | `false` | Output in JSON format |

## Examples

### List detected platforms

```bash title="Terminal"
npx firmis list
```

### JSON output for scripting

```bash title="Terminal"
npx firmis list --json
```
