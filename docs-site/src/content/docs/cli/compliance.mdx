## Usage

```bash title="Terminal"
firmis compliance [path] [options]
```

## What it does

Maps scan findings to compliance frameworks:

- **SOC 2** — security control evidence
- **EU AI Act** — AI-specific compliance requirements
- **GDPR** — data protection assessment
- **NIST AI RMF** — risk management framework mapping
- **OWASP LLM Top 10** — LLM-specific vulnerability mapping

## Options

| Flag | Type | Default | Description |
|---|---|---|---|
| `--framework <name>` | string | `all` | Specific framework: `soc2`, `ai-act`, `gdpr`, `nist`, `owasp` |
| `--format <type>` | enum | `html` | Output format: `html`, `json` |
| `--output <file>` | string | — | Save report to file |
| `--verbose` | boolean | `false` | Show detailed mapping |
| `--evidence` | boolean | `true` | Include evidence artifacts |
