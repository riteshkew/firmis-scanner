## Usage

```bash title="Terminal"
firmis monitor [options]
```

## What it does

The runtime monitor uses two channels to observe agent behavior:

- **Claude Code Hooks** — intercepts PreToolUse/PostToolUse events
- **MCP Proxy** — sits between client and server, inspecting all messages

Features:
- Blast radius scoring (0–100) with automatic response escalation
- Behavioral baseline learning
- Dangerous shell command detection
- Cross-channel event correlation

## Options

| Flag | Type | Default | Description |
|---|---|---|---|
| `--install-hooks` | boolean | `false` | Install Claude Code hooks |
| `--start-daemon` | boolean | `false` | Start the monitoring daemon |
| `--stop-daemon` | boolean | `false` | Stop the monitoring daemon |
| `--status` | boolean | `false` | Show monitor status |
| `--install` | boolean | `false` | Install all monitoring components |
| `--uninstall` | boolean | `false` | Remove all monitoring components |
