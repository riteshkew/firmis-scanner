## Threat categories

Firmis detects 16 categories of threats in AI agent code:

| Category | Examples | Severity Range |
|---|---|---|
| Tool Poisoning | Hidden instructions in tool descriptions | Critical |
| Data Exfiltration | Sending local data to external URLs | Critical–High |
| Credential Harvesting | Accessing AWS/GCP/Azure credentials | Critical–High |
| Prompt Injection | Instruction override in tool output | Critical–High |
| Secret Detection | Hardcoded API keys, tokens, passwords | Critical–Medium |
| Supply Chain | Known-vulnerable dependencies | High–Medium |
| Malware Signatures | Known malicious code patterns | Critical |
| Known Malicious | Packages flagged in threat databases | Critical |
| Network Abuse | Unauthorized DNS/HTTP requests | High–Medium |
| File System Abuse | Unauthorized file read/write | High–Medium |
| Permission Overgrant | Excessive tool permissions | High–Medium |
| Agent Memory Poisoning | Corrupting agent context/memory | High |
| Malware Distribution | Spreading malicious payloads | Critical–High |
| Privilege Escalation | Gaining unauthorized access | High |
| Insecure Configuration | Weak security settings | Medium–Low |
| Access Control | Missing authentication/authorization | High–Medium |
